## 개요
운전자가 현재 위치를 기준으로 가격이 가장 저렴한 주유소 TOP5를 지도와 함께 확인하고, 선택한 주유소까지 카카오맵으로 길찾기를 시작할 수 있다.

## 범위

### 포함
- 브라우저 위치 정보 기반 반경 검색 (3km→5km→10km 확장)
- 유종 선택 (휘발유/경유/LPG 중 단일 선택, 기본값 휘발유)
- 브랜드 다중 선택 필터, 셀프주유소 추정 필터
- 가격 오름차순(동률 시 거리 가까운 순) TOP5 리스트 + 지도 핀 표시, 리스트-지도 연동
- 리스트 항목에서 카카오맵 길찾기 연동
- PWA 홈 화면 설치

### 제외
- 전기차 충전소 가격 비교: 이용 가능한 공공 API에 충전 요금 필드가 없음 (`idea.md` 검증 완료)
- 자체 턴바이턴 내비게이션: 경로 계산·안내는 카카오맵에 위임
- 실시간 가격 알림/푸시: 오피넷 가격 데이터가 배치 주기로 갱신되어 실시간성이 낮음
- 로그인/회원가입: 위치 기반 조회에 계정이 필요 없음
- 수동 주소 입력 등 위치 폴백: 위치 권한 허용이 유일한 진입 경로이며, 거부 시 재시도만 제공 (MVP 범위 최소화)

## 시나리오

### S1. 최초 진입 및 위치 허용 → 결과 표시
- **Given**: 앱을 처음 열었고 위치 권한 프롬프트가 아직 뜨지 않은 상태, 유종은 휘발유가 기본 선택되어 있음
- **When**: 위치 권한 프롬프트에서 허용을 누른다
- **Then**
  - [x] S1-1 허용 직후 로딩 스피너와 "근처 주유소를 찾는 중..."(제안 기본값) 문구가 보인다
  - [x] S1-2 조회 완료 후 결과 리스트에 최대 5개 항목이 순위(1~5), 가격("1,830원" 형식), 거리(1km 미만은 "850m", 이상은 "2.9km"처럼 소수점 1자리) 형식으로 표시된다
  - [x] S1-3 리스트는 가격 오름차순으로 정렬되고, 가격이 같으면 거리가 가까운 순으로 정렬된다
  - [x] S1-4 지도에 현재 위치가 별도 마커로 표시되고, 리스트의 모든 주유소 핀이 화면 안에 들어오도록 자동으로 줌이 조정된다

### S2. 유종 변경
- **Given**: S1의 결과가 표시된 상태
- **When**: 유종 선택을 "경유"로 바꾼다
- **Then**
  - [x] S2 반경 3km부터 다시 조회되어 경유 기준 TOP5로 리스트와 지도가 즉시 갱신된다

### S3. 브랜드 필터
- **Given**: 결과가 표시된 상태, 브랜드 체크박스(SK에너지/GS칼텍스/HD현대오일뱅크/S-OIL/자가상표 등)가 모두 체크되어 있음
- **When**: 특정 브랜드 하나만 남기고 나머지 체크를 해제한다
- **Then**
  - [ ] S3 리스트와 지도가 즉시 갱신되어 선택한 브랜드의 주유소만 (반경 확장 로직을 다시 적용해) TOP5로 표시된다

### S4. 셀프주유소 필터
- **Given**: 결과가 표시된 상태, 셀프주유소 필터가 꺼져 있음(기본값)
- **When**: 셀프주유소 필터를 켠다
- **Then**
  - [ ] S4-1 주유소명에 "셀프"가 포함된 주유소만 리스트에 남아 TOP5로 갱신된다
  - [ ] S4-2 필터 토글 바로 아래에 "이름 기준 추정치이며 정확하지 않을 수 있어요"(제안 기본값) 안내 문구가 표시된다

### S5. 지도-리스트 연동
- **Given**: 결과 리스트가 표시된 상태
- **When**: 리스트에서 특정 주유소 항목을 탭한다
- **Then**
  - [x] S5 지도에서 해당 주유소의 핀이 강조 표시된다

### S6. 길찾기
- **Given**: 결과 리스트에 주유소 항목이 보이는 상태
- **When**: 항목의 "길찾기" 버튼을 누른다
- **Then**
  - [ ] S6 새 탭(또는 새 창)으로 현재 위치에서 해당 주유소까지의 카카오맵 길찾기 화면(앱 또는 웹)이 열린다

### S7. 반경을 확장해도 결과 부족
- **Given**: 특정 유종·브랜드·셀프 필터 조합에서 반경을 10km까지 확장해도 조건에 맞는 주유소가 5곳 미만임
- **When**: 조회가 완료된다
- **Then**
  - [x] S7-1 찾은 개수(0~4곳)만큼만 리스트와 지도에 표시된다
  - [ ] S7-2 1곳 이상이면 "10km 내 N곳만 찾았어요"(제안 기본값), 0곳이면 "10km 내에 조건에 맞는 주유소가 없어요"(제안 기본값) 안내 문구가 표시된다

### S8. 위치 권한 거부
- **Given**: 앱을 처음 열어 위치 권한 프롬프트가 뜬 상태
- **When**: 사용자가 권한 요청을 거부한다
- **Then**
  - [ ] S8 "위치 권한이 필요해요. 브라우저 설정에서 허용해주세요"(제안 기본값) 안내 문구와 "다시 시도" 버튼이 표시되고, 리스트와 지도는 빈 상태로 남는다

### S9. 오피넷 API 호출 실패
- **Given**: 위치 허용까지 완료되어 가격 정보를 조회하는 중
- **When**: 오피넷 API 호출이 실패하거나 타임아웃된다
- **Then**
  - [ ] S9 "가격 정보를 불러오지 못했어요. 다시 시도해주세요"(제안 기본값) 에러 메시지와 "다시 시도" 버튼이 표시된다

### S10. PWA 홈 화면 설치
- **Given**: 설치를 지원하는 브라우저로 앱에 접속한 상태
- **When**: 브라우저의 설치 프롬프트(또는 메뉴)를 통해 "홈 화면에 추가"를 실행한다
- **Then**
  - [ ] S10 홈 화면에 앱 아이콘이 생성되고, 아이콘 실행 시 브라우저 주소창 없이 독립 앱 화면으로 열린다

- [x] INV-1 리스트에 표시되는 주유소 수는 유종·브랜드·셀프 필터 조합과 무관하게 항상 5개를 넘지 않는다 (S1, S2, S3, S4, S7)

## End-to-end 검증

1. 앱 실행 후 위치 권한 프롬프트에서 거부 → 안내 문구와 "다시 시도" 버튼 확인 (S8)
2. "다시 시도" 클릭 후 이번에는 권한 허용 → 로딩 스피너 확인 → 결과 리스트(순위·가격·거리)와 지도(현재 위치 마커, 자동 줌) 확인 (S1)
3. 유종을 "경유"로 변경 → 리스트·지도가 갱신되는지 확인 (S2)
4. 브랜드 필터에서 한 브랜드만 남기고 나머지 해제 → 해당 브랜드만 남는지 확인 (S3)
5. 셀프주유소 필터 켜기 → 이름에 "셀프"가 포함된 항목만 남는지, 안내 문구가 뜨는지 확인 (S4)
6. 리스트 항목 하나를 탭 → 지도에서 해당 핀이 강조되는지 확인 (S5)
7. 해당 항목의 "길찾기" 클릭 → 새 탭에 카카오맵 길찾기가 열리는지 확인 (S6)
8. 브라우저 설치 메뉴로 "홈 화면에 추가" 실행 → 아이콘 생성과 독립 실행 확인 (S10)
9. (환경상 재현이 필요하면 별도로) 오피넷 API 응답을 인위적으로 실패시켜 에러 메시지·재시도 버튼 확인 (S9), 매우 외진 좌표로 조회해 10km 확장 후에도 5곳 미만인 경우의 안내 문구 확인 (S7)
10. 시나리오 전반에서 리스트 표시 개수가 5개를 넘지 않는지 확인 (INV-1)
